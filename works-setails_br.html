Sales Dashboard com Python

Data development - 2022

Este código é uma aplicação de visualização de dados que utiliza as bibliotecas pandas, plotly.express e streamlit. O objetivo da aplicação é exibir um painel de vendas com informações como vendas totais, avaliação média, vendas por linha de produto e vendas por hora. A aplicação lê um arquivo Excel e usa filtros na barra lateral para que o usuário possa selecionar a cidade, o tipo de cliente e o gênero para visualizar os dados correspondentes.

Por meio de gráficos de barras, pizza e linhas interativos, o usuário pode facilmente visualizar como as vendas são distribuídas por linha de produto e horário do dia, além de conferir informações importantes, como vendas totais e médias de avaliações de clientes. A aplicação é uma ferramenta útil para gerentes de vendas e tomadores de decisão, pois permite identificar rapidamente padrões e tendências nos dados de vendas.

Como o código funciona

O código começa importando as bibliotecas necessárias, como Pandas para manipulação de dados, Plotly e Graph_objs para criação de gráficos e Streamlit para criação de aplicativos web. Em seguida, é configurada a página do aplicativo com um título, ícone e layout.

A função "@st.cache" é usada para armazenar em cache o resultado da leitura de um arquivo do Excel, evitando a leitura repetitiva de dados não alterados.

Em seguida, são criados filtros de seleção de dados na barra lateral. Esses filtros permitem que o usuário selecione cidades, tipos de clientes e gênero. Depois, é executada uma consulta nos dados do Excel com base nesses filtros.

A página principal exibe três principais indicadores de desempenho, como vendas totais, classificação média e vendas médias por transação. Em seguida, o código cria dois gráficos de barras, um mostrando as vendas por linha de produto e outro mostrando as vendas por hora. Além disso, um gráfico de pizza e um gráfico de linhas são criados para visualização de dados. Os gráficos são plotados em duas colunas, com um gráfico em cada coluna.

Por fim, é definida uma função "show_st_style" para exibir o cabeçalho, rodapé e menu do Streamlit na página.

  • Vendas por Linha de Produto [Gráfico de Barras]: Este gráfico de barras mostra as vendas totais por linha de produto. Cada barra representa uma linha de produto, sendo que a altura da barra indica as vendas totais para aquela linha de produto. A cor das barras é consistente e representa um código de cor para todas as barras do gráfico.
  • Vendas por Hora [Gráfico de Barras]: Este gráfico de barras mostra as vendas totais para cada hora do dia. O eixo x representa as horas do dia, e o eixo y representa as vendas totais para cada hora. Cada barra representa uma hora, sendo que a altura da barra indica as vendas totais para aquela hora. A cor das barras é consistente e representa um código de cor para todas as barras do gráfico.
  • Vendas por Linha de Produto [Gráfico de Pizza]: Este gráfico de pizza mostra a porcentagem de vendas totais para cada linha de produto. Cada fatia representa uma linha de produto, sendo que o tamanho da fatia indica a porcentagem de vendas totais para aquela linha de produto. A legenda do lado direito do gráfico fornece informações sobre a cor de cada fatia.
  • Vendas por Hora [Gráfico de Linhas]: Este gráfico de linhas mostra as vendas totais para cada hora do dia. O eixo x representa as horas do dia, e o eixo y representa as vendas totais para cada hora. A linha representa as vendas totais para cada hora, sendo que a trajetória da linha indica a tendência geral de vendas ao longo do dia.

Os gráficos de barras são divididos em dois painéis, sendo o primeiro mostrando as vendas por hora e o segundo mostrando as vendas por linha de produto.
