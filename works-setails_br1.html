Survey Data Dashboard

Data development - 2022

Este código realiza a análise e visualização de resultados de uma pesquisa de satisfação armazenada em um arquivo Excel usando as bibliotecas Pandas, Streamlit e Plotly em Python. A biblioteca Streamlit cria uma interface interativa que permite ao usuário filtrar os resultados por departamento e faixa etária.

Um gráfico de barras é gerado para mostrar a distribuição das classificações de satisfação dos participantes, juntamente com uma imagem e uma tabela com os dados filtrados. Finalmente, um gráfico de pizza é plotado com o número total de participantes por departamento. O código permite que os usuários explorem e analisem os resultados da pesquisa com facilidade, tornando-o uma ferramenta útil para análise e relatórios de dados.

Como o código funciona

Este código é um exemplo de como analisar e visualizar os resultados de uma pesquisa de satisfação armazenada em um arquivo Excel usando Python. A biblioteca Pandas é usada para carregar os dados em um DataFrame. Em seguida, a biblioteca Streamlit é usada para criar uma interface interativa em que o usuário pode filtrar os resultados da pesquisa por departamento e faixa etária.

Um gráfico de barras é então gerado mostrando a distribuição das classificações de satisfação dos participantes, bem como uma imagem e uma tabela com os dados filtrados. Finalmente, um gráfico de pizza é plotado com o número total de participantes por departamento usando a biblioteca Plotly Express.

  • Carregando dados do arquivo Excel em um DataFrame usando Pandas
  • Criando a interface interativa com Streamlit para permitir a seleção dos filtros de pesquisa
  • Gerando os gráficos de barras e de pizza usando Plotly Express para visualizar os resultados da pesquisa.

Em resumo, este código oferece uma maneira fácil de visualizar os resultados de uma pesquisa de satisfação e permite que o usuário customize a análise com base em seus critérios de seleção.
