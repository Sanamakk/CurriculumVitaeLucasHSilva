Survey Data Dashboard

Data development - 2022

This code performs data analysis and visualization of satisfaction survey results stored in an Excel file using the Pandas, Streamlit, and Plotly libraries in Python. The Streamlit library creates an interactive interface that allows the user to filter the results by department and age range.

A bar chart is then generated to show the distribution of satisfaction ratings from participants, along with an image and a table with the filtered data. Finally, a pie chart is plotted with the total number of participants per department. The code enables users to explore and analyze survey results with ease, making it a useful tool for data analysis and reporting.

How the code works

This code is an example of how to analyze and visualize the results of a satisfaction survey stored in an Excel file using Python. The Pandas library is used to load the data into a DataFrame. Next, the Streamlit library is used to create an interactive interface where the user can filter the survey results by department and age range.

A bar chart is then generated showing the distribution of satisfaction ratings of the participants, as well as an image and a table with the filtered data. Finally, a pie chart is plotted with the total number of participants by department using the Plotly Express library.

  • Loading data from the Excel file into a DataFrame using Pandas
  • Creating the interactive interface with Streamlit to allow for selection of the search filters
  • Generating the bar and pie charts using Plotly Express to visualize the survey results.

In summary, this code offers an easy way to visualize the results of a satisfaction survey and allows the user to customize the analysis based on their selection criteria.
