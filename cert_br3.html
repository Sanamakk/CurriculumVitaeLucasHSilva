Analista de Dados

Data de conclusão - 2022

O curso de Analista de Dados oferecido pela Faculdade XP - IGTI foi um programa que me forneceu uma base sólida em análise de dados. O curso abordou vários tópicos, incluindo limpeza e preparação de dados, análise estatística, visualização de dados e algoritmos de aprendizado de máquina. O currículo foi projetado para equipar os alunos com as habilidades necessárias para ter sucesso como analista de dados em qualquer setor.

Um dos principais aprendizados do curso foi a importância da limpeza e preparação de dados. Aprendi como identificar e lidar com dados ausentes ou incorretos, o que é crucial para garantir que os resultados da análise sejam precisos e confiáveis. Além disso, aprendi como usar o SQL para gerenciar grandes bancos de dados relacionais e recuperar dados de maneira eficiente.

Outro aspecto crucial do curso foi aprender sobre análise estatística. Adquiri conhecimento sobre os diferentes tipos de variáveis estatísticas, como variáveis categóricas e contínuas, e como usar vários métodos estatísticos para analisar e interpretar dados. O curso também abordou ferramentas como R, Weka e Knime, amplamente usadas na indústria de análise de dados.

Três tópicos importantes abordados no curso:

  • Visualização de Dados: Este tópico me ensinou como apresentar dados de forma eficaz usando gráficos, tabelas e outras técnicas de visualização. Ao criar visuais claros e concisos, pude comunicar informações complexas aos interessados de maneira mais acessível e compreensível.
  • Algoritmos de Aprendizado de Máquina: Este tópico me apresentou a algoritmos de aprendizado de máquina e como eles podem ser usados para fazer previsões com base em dados. Aprendi como aplicar algoritmos como árvores de decisão, florestas aleatórias e redes neurais para resolver problemas do mundo real.
  • Ética de Dados: Este tópico explorou as considerações éticas envolvidas no manuseio e análise de dados. Aprendi sobre a importância de proteger informações pessoais e garantir que os resultados da análise sejam justos e imparciais.

Em geral, o curso de Analista de Dados oferecido pela Faculdade XP - IGTI foi um excelente investimento em meu desenvolvimento de carreira. Ele me forneceu as habilidades e conhecimentos necessários para ter sucesso como analista de dados e me deu confiança para lidar com projetos complexos com facilidade.
