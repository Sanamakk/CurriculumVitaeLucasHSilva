Ciência de Dados

Data de conclusão - 2022

O curso de Ciência de Dados oferecido pela Faculdade XP - IGTI foi um programa abrangente que cobriu uma ampla gama de tópicos, desde limpeza e preparação de dados até algoritmos de aprendizado de máquina e visualização de dados. O curso foi projetado para fornecer uma base sólida para uma carreira em ciência de dados, e eu pude aplicar as habilidades que aprendi em projetos do mundo real.

Uma das coisas mais valiosas que aprendi no curso foi como abordar um projeto de análise de dados de forma sistemática. Seguindo uma abordagem estruturada, eu pude quebrar problemas complexos em partes menores e mais gerenciáveis e aplicar as técnicas apropriadas a cada etapa da análise. Essa abordagem me ajudou a identificar potenciais problemas desde o início e garantir que estava trabalhando com dados precisos e confiáveis.

Além dos tópicos acima, o curso também abordou ferramentas e tecnologias essenciais, como Spark e PySpark. Essas ferramentas são cruciais para lidar com grandes conjuntos de dados e realizar computação distribuída, e o curso me forneceu uma compreensão sólida de como usá-las efetivamente.

Além disso, adquiri uma compreensão mais profunda de conceitos estatísticos, como testes de hipóteses, intervalos de confiança e análise de regressão. Esses conhecimentos são essenciais para desenvolver modelos precisos e tirar insights significativos dos dados.

Tópicos importantes abordados no curso:

  • Análise Exploratória de Dados: Este tópico me ensinou como explorar e entender os dados antes de construir qualquer modelo. Usando técnicas como visualização de dados e estatísticas descritivas, eu pude identificar padrões e tendências nos dados e descobrir possíveis problemas que precisavam ser abordados.
  • Engenharia de recursos: Este tópico me mostrou como criar novos recursos a partir de dados existentes para melhorar a precisão dos modelos de aprendizado de máquina. Compreendendo as relações subjacentes entre variáveis, eu pude projetar recursos mais preditivos da variável de resultado, resultando em modelos mais precisos.
  • Avaliação de modelo: Este tópico me ensinou como avaliar o desempenho dos modelos de aprendizado de máquina e compará-los entre si. Usando técnicas como validação cruzada e curvas ROC, eu pude determinar qual modelo era o melhor ajuste para um problema específico.

Em geral, o curso de Ciência de Dados oferecido pela Faculdade XP - IGTI me proporcionou uma compreensão abrangente de todo o processo de ciência de dados, desde a preparação de dados até a implementação de modelos. Com as habilidades e conhecimentos que adquiri no curso, estou confiante em minha capacidade de enfrentar qualquer projeto de ciência de dados com confiança e precisão.
