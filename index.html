Latest works

Sales Dashboard

This code is a sales visualization dashboard that uses the Streamlit library in conjunction with the Pandas and Plotly libraries. It loads data from an Excel file, allows filtering of the data based on specific criteria selected by the user in the sidebar, and then displays charts and statistics based on the filtered data. The dashboard presents four main charts: a sales table by product line, a bar chart of sales by the hour of the day, a pie chart of sales by product line, and a line chart of sales by the hour of the day. In addition, it displays three main KPIs: total sales, average rating, and average sales per transaction.

Survey Data Dashboard

A dashboard was created to visualize the survey results, which are stored in an Excel file. Streamlit is used to display an interactive interface that allows users to filter results by department and age range. The results are then presented in a bar chart, along with an image and table displaying the filtered data. Lastly, a pie chart is generated to display the total number of participants in each department.

Login/Registration with database

Flask application that manages the creation of notes for logged-in users. The application allows users to log in and create notes, displaying the user's previous notes when the user logs in again. The code includes four files, each with its own functionality. models.py defines the database models, views.py defines the routes that allow the user to view and delete notes, auth.py defines the routes for user login and registration, and init.py is responsible for initializing the Flask application and defining the database connection.
