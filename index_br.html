Trabalhos Mais Recentes

Sales Dashboard

Este código é um painel de visualização de vendas que utiliza a biblioteca Streamlit em conjunto com as bibliotecas Pandas e Plotly. Ele carrega dados de um arquivo Excel, permite a filtragem dos dados com base em critérios específicos selecionados pelo usuário na barra lateral e, em seguida, exibe gráficos e estatísticas com base nos dados filtrados. O painel apresenta quatro principais gráficos: uma tabela de vendas por linha de produto, um gráfico de barras de vendas por hora do dia, um gráfico de pizza de vendas por linha de produto e um gráfico de linha de vendas por hora do dia. Além disso, ele exibe três principais KPIs: vendas totais, classificação média e vendas médias por transação.

Login/Registration com database

Aplicativo Flask que gerencia a criação de notas para usuários logados. A aplicação permite que os usuários façam login e criem notas, exibindo as notas anteriores do usuário quando o usuário fizer login novamente. O código inclui quatro arquivos, cada um com sua própria funcionalidade. O models.py define os modelos do banco de dados, o views.py define as rotas que permitem que o usuário visualize as notas e as exclua, o auth.py define as rotas para o login e o cadastro do usuário e o init.py é responsável por inicializar o aplicativo Flask e definir a conexão com o banco de dados.
